Conseils de traduction

Voiçi quelques recommandations pour la traduction du Guide de l'utilisateur Haiku.
  • N’altérez pas le code HTML présent dans certains blocs de traduction.
  • Ecrivez en français courant : Ni vulgaire, ni trop sérieux.
  • Avant de traduire le titre d'une page, vérifiez la traduction utilisée sur la page d'index (“Contents”).
  • Ne traduisez pas le nom des applications.
  • Respectez les règles typographiques du français : la virgule, le point et le point-virgule sont suivis, mais non précédés, par une espace. Le deux-points, le point d'exclamation et le point d'interrogation sont précédés par une espace insécable et suivis par une espace. Les signes mathématiques sont précédés et suivis par une espace insécable.
  • Voici quelques traductions de termes couramment utilisés :
    • Desktop : Bureau
    • Drag&drop : glisser-déposer
    • Driver : Pilote
    • Haiku : Haiku (C'est un nom propre, donc on ne rajoute pas de tréma sur le “i”)
    • Kernel : Noyau
    • Query : Requête
    • Replicant : Réplicant
    • Tracker, Deskbar : Le Tracker, la Deskbar
  • En cas de doute sur la traduction, marquez votre traduction comme "possiblement incorrecte" (case "Fuzzy" depuis la fenêtre de traduction), et envoyez un message sur la liste haiku-i18n-fr pour en discuter avec les autres traducteurs.
